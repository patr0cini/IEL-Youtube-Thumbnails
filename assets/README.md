# /assets

Recursos gerais da aplicação.

## Logótipo

Coloca aqui o logótipo da igreja com o nome **`logo.png`**:

```
assets/logo.png
```

- Se o ficheiro existir, aparece automaticamente no **canto superior esquerdo**
  de cada thumbnail.
- Se não existir, é usado um **monograma "iel" embutido** (anel branco + "iel").
- Recomendado: PNG com fundo transparente, versão **clara/branca** (para
  contrastar com o fundo escuro), pelo menos 400px de largura.
