# /backgrounds

Fundos usados nas **mensagens isoladas** (sem série).

- Coloca aqui imagens `.png`, `.jpg`, `.jpeg` ou `.webp`.
- Resolução recomendada: **1280×720** (ou maior, com o mesmo rácio 16:9).
- Quando uma mensagem é isolada, a app escolhe **aleatoriamente** um destes fundos.

> Se esta pasta estiver vazia, a aplicação gera automaticamente fundos
> cinematográficos por código (degradés escuros + brilhos subtis), por isso
> funciona logo sem precisares de adicionar nada.

Para adicionar um fundo novo basta largar o ficheiro aqui e reiniciar o `npm run dev`.
